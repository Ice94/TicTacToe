# Tic Tac Toe Game 
This project is about simple game Tic Tac Toe. Main objective to achieve is full OOP design,
with good test coverage.
## 1. How to run the project
### 1.1 Installation

``mvn clean install``
### 1.2 Testing
``mvn test``
### 1.3 How to run
``mvn exec:java -Dexec.mainClass="com.bratek.Main"``
